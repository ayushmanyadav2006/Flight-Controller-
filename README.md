# Flight-Controller-

Created a electronic control hardware to control the roll, pitch, yaw of a airplane
Also the speed of the airplane, landing gears etc

The electronics consists of 
Arduino Uno (Microcontroller board) 
Joystick (Controlling the roll and pitch)
Potentiometer (Controlling the speed of the airplane)
Buttons (Controlling the yaw of the airplane) 
Joystick button (Controlling the landing gear) 

All the connections were made on breadboard by using jumper wires

I also made the PCB for the samne breadboard electronics.
